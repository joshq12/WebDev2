Welcome

Welcome to the Dog Breeds website. This site allows users to explore different dog breeds, view detailed information, and interact with dynamically generated content using JavaScript.

The website demonstrates the use of data structures, data manipulation, Handlebars templating, and interactive features across multiple pages.

What You Can Do

  • Browse popular dog breeds
  • View an interactive image gallery
  • Search, filter, and sort breed data
  • Add and remove dog breeds dynamically
  • View calculated statistics such as average lifespan
