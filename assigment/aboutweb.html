About This Webpage

This webpage is a simple pet database designed to provide information about different dog breeds. It includes features such as searching for breeds by name, filtering by friendliness, and viewing images of various dog breeds.

The webpage is built using HTML and CSS, with some JavaScript functionality for interactivity. It serves as a basic example of how to create a user-friendly interface for displaying information about pets.

Feel free to explore the different dog breeds and learn more about them!
